# 施設マップ

保育園、幼稚園、小学校などの施設をマップ上で検索・表示するWebアプリケーション。

## 機能

- **マップ表示**: Leaflet.jsを使用したインタラクティブなマップ
- **ピン検索**: マップ上にピンを指して、半径n km以内の施設を検索
- **フィルタリング**: 都道府県と施設種類で絞り込み
- **距離計算**: Haversine公式を使用した正確な距離計算
- **データ保存**: IndexedDBで検索結果をローカルに保存
- **複数都道府県対応**: 茨城、群馬、山梨、新潟、長野、栃木、富山

## 使い方

1. ブラウザで `index.html` を開く
2. マップ上の施設が表示されます
3. **ピンボタン** をクリックしてマップ上にピンを指す
4. 半径を設定して検索実行
5. 施設一覧から選択してGoogle Mapsで確認可能

## ファイル構成

```
.
├── index.html                  # メインアプリケーション
├── facilities_data/            # 施設データ
│   ├── 茨城_facilities.json
│   ├── 茨城_facilities.csv
│   ├── 群馬_facilities.json
│   ├── 群馬_facilities.csv
│   └── ... (他の都道府県)
└── README.md
```

## 技術スタック

- **フロントエンド**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **マップライブラリ**: Leaflet.js v1.9.4
- **タイルプロバイダー**: OpenStreetMap
- **データベース**: IndexedDB
- **データ形式**: JSON, CSV

## 必要環境

- モダンブラウザ（Chrome, Firefox, Safari, Edge等）
- インターネット接続（マップタイルの表示用）
